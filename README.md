## 🎣 一键运行 / Quick Play / 一鍵運行

```bash
# Web版 (推荐)
npm install && npm run dev
# http://localhost:5173

# Docker版
docker run -p 3000:3000 fishing-game

# Python版
pip install -r requirements.txt
python main.py
```
